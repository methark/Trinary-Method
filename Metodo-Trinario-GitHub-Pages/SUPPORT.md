# Apoie o Método Trinário | Support the Trinary Method

O projeto busca apoio para pesquisa matemática, revisão acadêmica, implementação computacional, documentação bilíngue, testes e publicação científica.

Formas de apoio:

- divulgar o repositório e citar o trabalho;
- oferecer revisão técnica ou acadêmica;
- contribuir com testes e estudos comparativos;
- financiar bolsas, infraestrutura, publicação e revisão independente;
- patrocinar marcos públicos do roadmap.

## GitHub Sponsors

Para ativar o botão **Sponsor**, o proprietário deve se inscrever no GitHub Sponsors e configurar um perfil de recebimento. Após a aprovação, inclua o usuário em `.github/FUNDING.yml`.

## English

Support helps fund mathematical research, academic review, reference implementations, bilingual documentation, testing, and scientific publication. You may share and cite the project, contribute qualified review, sponsor roadmap milestones, or fund independent validation.
