# Contribuição | Contributing

## Português

1. Abra uma issue descrevendo a proposta antes de mudanças extensas.
2. Faça um fork e crie uma branch específica.
3. Preserve a autoria e cite fontes acadêmicas e empíricas.
4. Envie um pull request pequeno, verificável e bilíngue quando aplicável.
5. Nenhuma contribuição é incorporada sem revisão de um mantenedor admitido.
6. Acesso de escrita direta é reservado a colaboradores explicitamente convidados.

Contribuições podem ser recusadas por desalinhamento de escopo, ausência de evidências, falta de reprodutibilidade ou risco à integridade conceitual do projeto.

## English

1. Open an issue before proposing extensive changes.
2. Fork the repository and create a focused branch.
3. Preserve authorship and cite academic and empirical sources.
4. Submit a small, verifiable pull request, bilingual when applicable.
5. No contribution is merged without review by an admitted maintainer.
6. Direct write access is reserved for explicitly invited collaborators.

Contributions may be declined for scope misalignment, missing evidence, lack of reproducibility, or risk to the project's conceptual integrity.
