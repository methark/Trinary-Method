# Governança | Governance

## Modelo

Arthur Braga Padilha é o autor e mantenedor fundador. Decisões sobre semântica nuclear, nomenclatura, versões oficiais e direção científica exigem sua aprovação ou a aprovação de mantenedores por ele designados.

## Admissão

Colaboradores com permissão de escrita são admitidos por convite. Participação em issues ou pull requests não concede automaticamente acesso ao repositório.

## Fluxo de mudanças

- alterações entram por pull request;
- a branch principal deve permanecer protegida;
- pelo menos uma aprovação de mantenedor é exigida;
- conversas devem ser resolvidas antes do merge;
- force push e exclusão da branch principal devem permanecer bloqueados;
- mudanças formais relevantes devem incluir justificativa e testes.

## English summary

Arthur Braga Padilha is the founding author and maintainer. Write access is invitation-only. Official changes must use pull requests, maintainer review, protected branches, resolved conversations, and evidence or tests for material formal changes.
