# Método Trinário | Trinary Method

[Português](#português) · [English](#english)

## Português

O **Método Trinário**, de Arthur Braga Padilha, é uma proposta de raciocínio epistêmico, graduado, contextual, diagnóstico e revisável. Em vez de forçar toda conclusão para “verdadeiro” ou “falso”, o método representa sustentação, refutação e indeterminação, incluindo a causa da suspensão do julgamento.

### Objetivos do projeto

- desenvolver uma especificação matemática verificável;
- construir uma implementação computacional de referência;
- comparar o método com lógicas binárias, fuzzy, trivalentes e paraconsistentes;
- criar conjuntos de testes reproduzíveis;
- estudar aplicações em IA, ciência, medicina, direito, robótica e decisão sob incerteza.

## English

The **Trinary Method**, authored by Arthur Braga Padilha, is a proposal for epistemic, graded, contextual, diagnostic, and revisable reasoning. Rather than forcing every conclusion into “true” or “false,” it represents support, refutation, and indeterminacy, together with the reason judgment was suspended.

### Project goals

- develop a verifiable mathematical specification;
- build a reference computational implementation;
- compare the method with binary, fuzzy, three-valued, and paraconsistent logics;
- create reproducible test suites;
- study applications in AI, science, medicine, law, robotics, and decision-making under uncertainty.
