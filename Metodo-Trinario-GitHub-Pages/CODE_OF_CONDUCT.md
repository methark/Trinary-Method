# Código de Conduta | Code of Conduct

Participantes devem agir com respeito, rigor intelectual e boa-fé. Críticas devem se concentrar em ideias, evidências, definições e resultados reproduzíveis. Assédio, ataques pessoais, discriminação, spam e manipulação de autoria não serão tolerados. Mantenedores podem editar, ocultar ou remover conteúdo e restringir participantes para proteger a comunidade e o projeto.
